出典: フリー百科事典『ウィキペディア（Wikipedia）』 移動: ナビゲーション, 検索 『ブレイキング・ザ・コード』（原題:Breaking The Code）は、アンドリュー・ホッジ（Andrew Hodges）による『Alan Turing:The Enigma』に基づいたヒュー・ホワイトモア作のイギリスの戯曲。実在のイギリス人数学者アラン・チューリングの暗号理論と同性愛の葛藤を描いている。初演は1986年。ブロードウェイではデレク・ジャコビ主演で1987年11月15日から1988年4月10日にかけて上演され、トニー賞3部門、ドラマ・デスク賞（Drama Desk Award）2部門にノミネートされた。1996年にはデレク・ジャコビ主演で、BBCでテレビドラマ化され、ブロードキャスティング・プレス・ギルド賞（Broadcasting Press Guild）を受賞した他、英国アカデミー賞、GLAAD賞（Gay & Lesbian Alliance Against Defamation）にノミネートされた。日本では、1988年に劇団四季によって、創立35周年記念として上演される。目次 1 オリジナル公演1.1 スタッフ  2 劇団四季公演2.1 スタッフ2.2 主要キャスト2.3 上演記録  3 外部リンク    [編集] オリジナル公演[編集] スタッフ台本 … ヒュー・ホワイトモア原作 … アンドリュー・ホッジ『Alan Turing:The Enigma』 [編集] 劇団四季公演[編集] スタッフ訳 … 吉田美枝演出 … 浅利慶太装置 … 朝倉摂照明 … 沢田祐二衣装 … 岸井克己音楽 … 鎮守めぐみ [編集] 主要キャストアラン・チューリング … 日下武史 [編集] 上演記録1988年9月2日〜9月8日 - 銀座セゾン劇場（東京初演）1988年10月10日〜10月16日 - 銀座セゾン劇場（東京再演）1990年10月20日〜11月16日 - 全国公演（全国各地で初演） [編集] 外部リンクBreaking the Code at the Internet Broadway DatabaseBreaking the Code - Internet Movie Database （英語）Breaking the Code at the Alan Turing Scrapbook この「ブレイキング・ザ・コード」は、舞台芸術に関連した書きかけ項目です。この記事を加筆・訂正などして下さる協力者を求めています（Portal:舞台芸術）。   「http://ja.wikipedia.org/wiki/%E3%83%96%E3%83%AC%E3%82%A4%E3%82%AD%E3%83%B3%E3%82%B0%E3%83%BB%E3%82%B6%E3%83%BB%E3%82%B3%E3%83%BC%E3%83%89」より作成カテゴリ: イギリスの戯曲 | 1980年代の戯曲 | 劇団四季の上演作品隠しカテゴリ: 舞台芸術関連のスタブ
