出典: フリー百科事典『ウィキペディア（Wikipedia）』 移動: ナビゲーション, 検索 恒等式（こうとうしき、identity）とは、等式すなわち等号 (=) を含む数式であって、そこに現れるあらゆる変数がどのような値にあっても、常に等号で結ばれた左右二つの数式の "値" が等しいもののことを言う。重要な恒等式の中には、公式と呼ばれて知られているものも多く存在する。オイラーの公式などはその一例である。[編集] 例次の式は x, y について恒等式である。 x2 + 2xy + y2 = (x + y)2. (1) が（少なくとも 3 つの値をとるような変数）x について恒等式であるとき、(2) が成立する ax2 + bx + c = 0 … (1),a = b = c = 0 … (2). 三角関数は次のような恒等式で結ばれている。 sin2(x) + cos2(x) = 1,tan(x) = sin(x) / cos(x). 1 = 1 はあらゆる変数に関する恒等式である。数の 2 は恒等式 2 = 1 + 1 によって定義される自然数である。ただし、右辺は「自然数 1 の次の数（後継 successor）である自然数」の意。 [編集] 関連項目ウィキブックスに恒等式関連の教科書や解説書があります。 数式等式方程式不等式  「http://ja.wikipedia.org/wiki/%E6%81%92%E7%AD%89%E5%BC%8F」より作成カテゴリ: 数式 | 数学に関する記事
